var/const_api.php - файл содержит необходимые константы, которые нужно натроить для работы ядра.
В папках лежат различные модули, которые можно подключать к сценариям и использовать необходимые фунции. 

Можно добавлять только универсальные функции, которые можно будет использовать на других сайтах.
Отделять функции обработки информации от функций генерации HTML кода! 
Отсылать другим разработчикам последние версии файлов, указывая при этом на сделанные изменения в этом файле.
